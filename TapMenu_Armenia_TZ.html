01 Обзор проекта

TapMenu Armenia — SaaS-платформа для ресторанов Армении. Гость прикладывает смартфон к NFC-метке на столе → мгновенно открывается веб-меню ресторана на его языке. Без установки приложений, без регистрации.

👤 Для гостя

Одно касание → меню. Выбор языка, просмотр блюд, детали и аллергены, переключение валюты.

🍽 Для ресторана

Личный кабинет: управление меню, библиотека блюд, аналитика, специальные предложения, настройка шаблона.

⚙️ Для суперадмина

Управление всеми ресторанами, подписками, заявками с сайта, курсами валют и настройками платформы.

02 Технический стек

⚠️
Единый стек для всего проекта. В предыдущих документах были противоречия (Node.js vs Next.js, AWS vs Supabase). Этот документ — окончательный эталон. Все решения ниже обязательны.

Frontend

  • ✦ Next.js 14 (App Router)
  • ✦ TypeScript (strict mode)
  • ✦ Tailwind CSS + shadcn/ui
  • ✦ Framer Motion (анимации)
  • ✦ Zustand (state management)
  • ✦ TanStack Query (API кэш)
  • ✦ React Hook Form + Zod
  • ✦ @dnd-kit/core (drag & drop)

Backend & Database

  • ✦ Next.js API Routes + Server Actions
  • ✦ Supabase (PostgreSQL + Auth + Storage)
  • ✦ Supabase Row Level Security (RLS)
  • ✦ Supabase JS Client (типы из схемы БД — вместо Prisma)
  • ✦ Vercel Cron Jobs (фоновые задачи)
  • ✦ Resend (email уведомления)
  • ✦ Cloudflare Images — загрузка, трансформация, CDN (замена Sharp.js)

Внешние интеграции — MVP

  • ✦ Yandex Cloud Translate API — все тарифы (RU/EN/HY + 100 языков)
  • ✦ DeepL API — расширенный перевод (LUXE, fallback для редких языков)
  • ✦ ЦБ РА API — курсы валют (бесплатно)
  • ✦ OpenAI GPT-4o + DALL-E 3 — только LUXE

Платежи — Phase 2 (не MVP)

  • ⏳ Stripe — карты Visa / Mastercard
  • ⏳ IDram — армянская платёжная система
  • ⏳ Годовые подписки со скидкой 20%
ℹ️
Хостинг: Vercel (автодеплой из GitHub). Изображения — Cloudflare Images (загрузка + CDN + трансформация). Не использовать AWS в MVP.
⚠️
Важно: Supabase JS Client вместо Prisma ORM. Supabase автоматически генерирует TypeScript-типы из схемы БД (supabase gen types typescript). Использование Prisma обходит Row Level Security — это критическая дыра в безопасности. Все запросы к БД — только через Supabase JS Client с JWT-токеном пользователя.

🔤
Почему Yandex Cloud Translate? Единственный сервис, который отлично работает сразу с тремя ключевыми языками проекта: Армянский (HY) — которого нет в DeepL, Русский — лучшее качество среди всех сервисов, Английский. Поддерживает 100+ языков. Цена: ~$5 за 1M символов, есть бесплатный tier.

DeepL остаётся как опциональный fallback только для LUXE тарифа при запросе редких европейских языков.

08 Модуль 3 — Кабинет ресторана (app.tapmenu.am)

🔐
Самостоятельная регистрация не предусмотрена. Аккаунт создаёт суперадмин. После создания на email ресторана приходит письмо с логином и временным паролем. При первом входе система обязывает сменить пароль.

8.1 Аутентификация

  • 📧 Страница входа: Email + Пароль (Supabase Auth)
  • 🔑 Восстановление пароля через email
  • 🔄 При первом входе — обязательная смена временного пароля
  • 🚀 Онбординг-визард (3 шага): загрузка логотипа → описание ресторана → добавление первого блюда

8.2 Главный дашборд

Быстрая статистика (сегодня): 👁 Просмотров: 142 · 🍽 Блюд в меню: 67

Языки гостей: 🇷🇺 60% · 🇬🇧 30% · 🇦🇲 10%

🍽 Топ-5 популярных блюд (Pro/Premium/LUXE)

8.3 Управление меню

Список всех меню ресторана. Создать новое (в рамках лимита тарифа). Только одно меню может быть активным одновременно.

Редактор меню — двухпанельный интерфейс:

Левая панель — Категории
  • Drag & Drop сортировка
  • Создать категорию: название RU → "Перевести" → EN/HY
  • Выбор иконки (эмодзи)
  • Скрыть / показать категорию
  • Удалить (с предупреждением если есть блюда)
Правая панель — Блюда категории
  • Список блюд с toggle "в меню" и toggle "доступно (стоп-лист)"
  • Drag & Drop сортировка
  • Кнопка "Добавить из библиотеки"
  • Кнопка "Создать новое блюдо"

8.4 Библиотека блюд

📚
Отдельный раздел кабинета. Все блюда ресторана хранятся здесь. Каждое блюдо можно добавить в одну или несколько категорий меню или убрать оттуда — блюдо из библиотеки не удаляется.

Форма создания/редактирования блюда:

  • 📸 Фото — drag & drop, любой формат (JPG/PNG/HEIC/WebP) → автоконвертация в WebP (см. раздел 11)
  • 📝 Название RU + кнопка "Перевести" → EN, HY (и другие языки по тарифу)
  • 📄 Описание RU + кнопка "Перевести" → EN, HY
  • 💰 Цена (AMD)
  • ⚖️ Вес (граммы) — обязательное поле для отображения
  • 🔥 Калории (ккал) — необязательно, скрыто в меню
  • 🥩 Белки / Жиры / Углеводы — необязательно, скрыто в меню
  • ⚠️ Аллергены — мультивыбор из предустановленного списка, скрыто в меню
  • 🏷 Теги — Хит / Веган / Острое / Шеф рекомендует / Новинка, скрыто в меню

📶 Как найти SSID и пароль Wi-Fi? ▼ нажмите для инструкции

🔒 Пароль хранится в зашифрованном виде (AES-256). Никто, кроме системы, его не видит.
